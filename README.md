# How to use? 
1. Make a .env file in the directory and `PASSWORD=<your_password>'
2. `data/template.png` will contain template picture for generation
3. `EMAIL_ADDRESS` variable in emails.py will be your email
3. Subject and message content are inside emails.py, you might need to change them
4. inside `generator.py`, there is y_coordinate, data/data.csv contains the csv file (default to student data)
5. fonts inside `data/font.ttf`
